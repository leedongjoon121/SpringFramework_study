# HttpServletRequest
- HttpServletRequest로 부터 url를 통한 값을 가져오고
- Model 객체에 데이터를 전송하는 방식
- url로 부터 값이 넘어오지 않아도 페이지가 넘어가긴 한다.

# RequestParam
- url로 부터 넘어오는 값을 미리 할당해 놓고 그 값을 Model객체로 전송하는 방식
- url로 부터 값이 넘어오지 않으면 에러발생
